# StudentHub
A Java desktop application that stimulates a student hub system.

##Features:
- Queue-based student management
- Grouping students by area (Cafe, Lounge, Study Room)
- Sorting bby name and bill amount using lambda expressions
- Recursive total profit calculation
- File saving and loading
- Swing GUI interface

##Technologies:
- Java
- Swing
- Collections
- File I/O
- OOP principles

## How to Run 
1. Clone or download this repository
2. Open the project in a Java IDE (IntelliJ, Eclipse, or VS code)
3. Make sure Java is installed (I reccomend JDK 17+)
4. Run the StudentHubGUI class
